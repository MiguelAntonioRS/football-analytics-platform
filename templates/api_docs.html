Documentación de la API

Base URL: http://localhost:8000/api/

Gestión de Ligas

GET /api/leagues/

Lista todas las ligas. Soporta paginación, búsqueda y ordenamiento.

Parámetros:

  • search - Buscar por nombre, país o temporada
  • ordering - Ordenar por nombre, país, temporada o fecha
  • page - Número de página
POST /api/leagues/

Crea una nueva liga.

Campos requeridos: name, country, season

GET /api/leagues/{id}/

Obtiene los detalles de una liga específica.

PUT /api/leagues/{id}/

Actualiza una liga existente.

DELETE /api/leagues/{id}/

Elimina una liga.

Gestión de Equipos

GET /api/teams/

Lista todos los equipos. Soporta búsqueda y ordenamiento.

Parámetros:

  • search - Buscar por nombre, estadio o entrenador
  • ordering - Ordenar por nombre o fecha
POST /api/teams/

Crea un nuevo equipo.

Campos requeridos: name, league, stadium, coach

Gestión de Jugadores

GET /api/players/

Lista todos los jugadores.

Parámetros:

  • team - Filtrar por equipo
  • league - Filtrar por liga
  • position - Filtrar por posición
  • is_active - Filtrar por estado activo
POST /api/players/

Crea un nuevo jugador.

Campos requeridos: name, team, position, nationality

Gestión de Partidos

GET /api/matches/

Lista todos los partidos.

Parámetros:

  • league - Filtrar por liga
  • team - Filtrar por equipo
  • status - Filtrar por estado
  • from_date - Fecha inicial
  • to_date - Fecha final
POST /api/matches/

Crea un nuevo partido.

Campos requeridos: league, home_team, away_team, date

Eventos del Partido

GET /api/matches/events/

Lista todos los eventos.

Parámetros:

  • match - Filtrar por partido
  • event_type - Filtrar por tipo (goal, assist, shot, etc.)
  • player - Filtrar por jugador
  • team - Filtrar por equipo
POST /api/matches/events/

Crea un nuevo evento.

Campos requeridos: match, minute, event_type, player, team

Tipos de evento:

  • goal - Gol
  • assist - Asistencia
  • shot - Tiro
  • pass - Pase
  • foul - Falta
  • yellow_card - Tarjeta Amarilla
  • red_card - Tarjeta Roja
  • tackle - Entrada

Análisis y Estadísticas

GET /api/analytics/dashboard/

Obtiene datos del dashboard: resumen general, goleadores, asistentes y partidos recientes.

GET /api/analytics/teams/{id}/

Obtiene estadísticas detalladas de un equipo.

GET /api/analytics/players/{id}/

Obtiene estadísticas detalladas de un jugador.

GET /api/analytics/scorers/

Obtiene ranking de goleadores.

Parámetros:

  • limit - Número de resultados (default: 10)
  • league - Filtrar por liga
  • season - Filtrar por temporada
GET /api/analytics/assists/

Obtiene ranking de asistentes.

GET /api/analytics/leagues/{id}/

Obtiene estadísticas de una liga.

Reportes

GET /reports/team-comparison/

Compara dos equipos.

Parámetros:

  • team1 - ID del primer equipo
  • team2 - ID del segundo equipo
GET /reports/player-comparison/

Compara dos jugadores.

Parámetros:

  • player1 - ID del primer jugador
  • player2 - ID del segundo jugador
GET /reports/standings/{league_id}/

Obtiene la tabla de clasificación de una liga.

GET /reports/goals-by-player/

Obtiene ranking de goles por jugador.

GET /reports/goals-by-team/

Obtiene ranking de goles por equipo.

Ejemplos de Uso

cURL
# Listar todas las ligas
curl -X GET http://localhost:8000/api/leagues/

# Crear una liga
curl -X POST http://localhost:8000/api/leagues/ \
  -H "Content-Type: application/json" \
  -d '{"name": "La Liga", "country": "España", "season": "2024-2025"}'

# Listar partidos de una liga
curl -X GET "http://localhost:8000/api/matches/?league=1"

# Obtener top goleadores
curl -X GET "http://localhost:8000/api/analytics/scorers/?limit=5"
